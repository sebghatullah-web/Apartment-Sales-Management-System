🏢 Apartment Management System (PHP MVC)
This project is a complete Apartment Management System built using PHP (Custom MVC), MySQL, Bootstrap 5, and Chart.js.
It provides a full-featured platform for managing real estate projects, apartments, customers, reservations, sales, and financial payments.

🚀 Features
🔐 Authentication & Security
User login system

Role-Based Access Control (Admin, Staff, Viewer)

User registration

Change password

Secure session handling

Logout system

🧑‍💼 User & Activity Management
Activity Log (tracks all user actions)

User roles and permissions

Admin-only sections

🏗 Project & Property Management
Manage Projects

Manage Blocks

Manage Floors

Manage Apartments

Apartment availability status

Ownership (Sales) management

Reservation system

💰 Financial Management
Record customer payments

Track total paid, remaining balance

Printable payment invoice

Monthly sales and payment analytics

📊 Dashboard & Analytics

Clean and modern design
